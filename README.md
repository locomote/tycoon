Airline Tycoon
=======================

Expand your Airline to be the ultimate Tycoon!

## Install

```
npm install
bundle install
npm install -g gulp
```

## Develop

```
npm start
```

Includes
- Live reloading for both CSS and Javascript changes
- `.cjsx`: JSX in Coffeescript

`gulp build` to create minified versions for prod.
